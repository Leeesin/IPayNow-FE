# 易售云后台管理系统

# 如何运行

TIP

  1. git clone git@code.ipaynow.cn:yishouyun/yishouyun-merchant-center-frontend.git
  2. 运行 yarn安装全部依赖
  3. 在根目录运行 yarn run dev

# 主要文件

注意

  • src/views/merchant-system: 商户后台
  • src/views/shop-system: 门店后台
  • src/components: 公共组件
  • src/plugin: vue插件
  • src/API: 环境变量

# 开发注意事项

注意

  • 生产环境分支为master,开发时需从此分支拉取代码。
  • 测试环境分支为dev
