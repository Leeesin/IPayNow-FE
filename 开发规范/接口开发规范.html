# 接口开发规范

TIP

  • 字段类型设置默认值，比如字段不存在，不能返回null或者删除字段，必须返回空字符串""或者空对象{}或者空数组[]
  • 列表查询 字段名统一为 data：{ list:[ ], total :"" }
  • 枚举值 后端返回 前端不做转换 文本翻译后端处理
  • 下拉列表数据结构与字段名统一为 { id :"文本1", name :"1"}
